dot-star
========

dot-star is a [Dots and Boxes](https://en.wikipedia.org/wiki/Dots_and_Boxes) web-based 2-player peer-to-peer game.


# Requirement
Tested to be working on Firefox 25.

# Technically
This game uses
* webRTC technology
* [jQuery](jquery.com)
* [p2p](https://github.com/js-platform/p2p)
